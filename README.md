# flexit

Install using mpm: 

```
npm i @yogevh83/flexit
```

Install using yarn:

```
yarn add @yogevh83/flexit
```

Usage:
```
import { Flexit } from "@yogevh83/flexit";
import "@yogevh83/flexit/dist-css/flexit.css";

/* Your Code Here */

<Flexit filterd={elementsToShowArray}>{elements}</Flexit>
```
